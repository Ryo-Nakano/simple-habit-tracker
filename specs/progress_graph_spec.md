# タスク進捗グラフ表示

## 概要
- 日次のタスク達成度をプログレスバーで視覚的に表示する。
- 4つのタスクのうち2つ完了していれば「50%」のように表示し、バーが半分まで伸びる形式とする。

## 仕様
### 機能要件
- 選択されている日付（`selectedDate`）のタスク達成度を計算し、表示する。
- 計算式: (完了タスク数 / 総タスク数) * 100
- タスク完了時にリアルタイムでバーの長さとパーセンテージが更新されること。

### UI/メッセージ
- **表示タイプ**: プログレスバー（横棒グラフ）。円グラフは使用しない。
- **配置場所**: `TaskList`コンポーネント内。
    - 具体的には、日付とカウントが表示されているヘッダー行（`<h2>...</h2>` と `<span>...</span>` のある行）の**直下**。
    - 「タスク追加フォーム（`AddTaskForm`）」の**直上**。
- **デザイン**:
    - 全幅（width: 100%）またはコンテナ幅に合わせたバー。
    - 完了部分: 緑色（`bg-green-500` 等）。
    - 背景部分: 薄いグレー（`bg-gray-200` 等）。
    - 角丸（rounded）処理を行い、柔らかな印象にする。
    - パーセンテージ数値（例: "50%"）はバーの中、またはバーの近くに配置し、視認性を確保する。既存の「2/4」表記と併用するか、置換するかはバランスを見て調整するが、グラフによる直感的な把握を優先する。

### 制約・前提
- 既存の `TaskList.tsx` を修正する。
- 新たなライブラリは導入せず、Tailwind CSS と標準の HTML/React で実装する。

## 実装計画
### 使用するクラス・ファイル
- `src/client/components/TaskList.tsx`

### 処理フロー
1. `TaskList` コンポーネント内で達成率を計算。
2. ヘッダー直下の `div` 要素としてプログレスバーを追加。
3. `style={{ width: \`\${percentage}%\` }}` を適用し、動的に幅を変更。
4. CSS transition (`transition-all duration-300` 等) を付与し、変化をアニメーションさせる。
